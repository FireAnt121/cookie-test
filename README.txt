== Features ==

- capture all query params in an http request (use case is UTM parameters in a facebook campaign)
- capture first session page URL and referrer
- store all this in cookies
- allow accessing all this data later using shortcodes e.g. to print one of the values stored in cookies
- allow injecting code based on if conditions on the values of the parameters e.g. if utm_source=facebook inject this code
